Para hacer funcionar EvenTag es necesario tener:

-Composer
-Laravel
-Node js
-XAMPP

Estas tres cosas son fundamentales para el correcto funcionamiento, por otro lado, las demas librerias implementadas se encuentran correctamente instaladas en la carpeta del proyecto o con CDN tales como (Tailwind, Sweet Alerts, Chart Js, Simple Software IO, etc...)

Por otro lado, es indispensable importar la base de datos MySQL que lleva por nombre "eventag", si le cambias el nombre a la BD recuerda cambiarlo tambien en el archivo .ENV

Para ejecutar el sistema tienes que abrir la carpeta del proyecto y en terminal git bash escribir "npm run dev" o hacerlo desde la terminal de Visual Studio Code o del editor de texto que utilices.
